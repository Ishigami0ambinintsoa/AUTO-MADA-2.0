BIENVENUE sur AUTO-MADA

AUTO- MADA est une application web destinée à l’apprentissage du code de la route. Elle guide les futurs conducteurs dans l’acquisition des règles de circulation, la compréhension des panneaux de signalisation, et la maître des règles de priorité.
L’ application permet également de se préparer à l’ examen théorique, de manière autonome et interactive.

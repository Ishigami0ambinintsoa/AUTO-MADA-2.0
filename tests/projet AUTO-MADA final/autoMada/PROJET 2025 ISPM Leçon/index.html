SITE pour apprendre la signification des panneaux de signalisation

I. Panneaux d'indication

Nom :Parking

Description :Ce panneau indique aux conducteurs qu'ils peuvent se garer à cet endroit.

Nom :Circulation à sens unique

Description :Ce panneau indique que la voie est en sens unique, c'est à dire que les véhicules ne peuvent circuler que dans une seule direction.

Nom :Emplacement d'arrêt d'urgence

Description :Ce panneau signale un emplacement spécialement aménagé sur l'autoroute ou une voie rapide, où les véhicules peuvent s'arrêter en cas d'urgence (par exemple: en panne, malaise, crevaison, etc...).

Nom :Priorité par rapport à la circulation venant en sens inverse

Description :Ce panneau indique que vous avez la priorité pour passer avant les véhicules venant en face, dans une zone de circulation alternée.

Nom :Entrée d'un tunnel

Description :Il signale aux conducteurs qu'ils vont entrer dans un tunnel, ce qui peutn être impliquer certaines règles particulières:

  • Allumage obligatoire des feux.
  • Interdiction de faire demi-tour ou de s'arrêter sauf urgence.
  • Respect des distances de sécurité.

II. Panneaux d'interdiction

Nom :Sens interdit

Description :Ce panneau interdit l'entrée à tous les vehicules dans une rue ou une voie, dans le sens où il est placé.

Nom :Stationnement interdit

Description :ce panneau signifie que le stationnement est interdit sur le côté de la route où le panneau est implanté.

Nom :Interdiction de faire demi-tour

Description :ce panneau signifie que les conducteurs ne peuvent pas faire demi-tour sur la route où on trouve l'implantation de cette panneau

Nom :interdiction de tourner à gauche

Description :Ce panneau signifie qu'il est interdit aux conducteur de tourner à gauche à l'intersection ou à l'endroit indiqué par la flèche du panneau.

Nom :interdiction de tourner à droite

Description :Ce panneau signifie qu'il est interdit aux conducteur de tourner à droite à l'intersection ou à l'endroit indiqué par la flèche du panneau.
